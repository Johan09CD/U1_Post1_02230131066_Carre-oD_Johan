Descripción

Proyecto correspondiente al laboratorio de la Unidad 1 Ecosistema Móvil y Fundamentos Avanzados. Se instaló y configuró Android Studio, se creó un proyecto en Kotlin llamado EcosistemaMovil, se ejecutó en un emulador y se utilizaron herramientas de depuración como Logcat y breakpoints.

Requisitos

Android Studio versión reciente
SDK Platform API 34 o superior
Emulador configurado
Conexión a internet para sincronización de Gradle

Ejecución

Abrir el proyecto en Android Studio.
Esperar la sincronización de Gradle hasta que aparezca BUILD SUCCESSFUL.
Iniciar el emulador desde Device Manager.
Ejecutar la aplicación con el botón Run.

Funcionalidad

La aplicación muestra información sobre tres paradigmas de desarrollo móvil Nativo, Híbrido y Multiplataforma mediante una interfaz construida dinámicamente en Kotlin.

Se implementaron mensajes de registro utilizando Log.d y Log.i y se realizó depuración colocando un breakpoint en el ciclo que recorre la lista de paradigmas para inspeccionar las variables en el Debugger.

Evidencias

El repositorio incluye el proyecto completo y las seis capturas solicitadas en la guía: SDK Manager, estructura del proyecto, ejecución en emulador con Hello World, aplicación con tarjetas de paradigmas, Logcat filtrado y Debugger detenido en breakpoint.
